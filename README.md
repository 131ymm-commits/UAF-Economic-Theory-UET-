# UET – Time‑Accounted Unified Economic Framework

**Единая теория экономики времени** на базе UAF (Universal Accounting Field).  
Версия: 1.0 | Статус: рабочая гипотеза | Лицензия: MIT

## 🧠 Основная идея

> Время — единственный фундаментально ограниченный и необратимый ресурс.  
> Деньги, товары, капитал, внимание, энергия — производные от времени.

**Уравнение консервации времени в замкнутой системе**:  
`t_past + t_now + t_future = const`

- **t_past** – кристаллизованное время (материальные блага, инфраструктура, знания)
- **t_now** – текущее действие и восприятие
- **t_future** – обещанное/ожидаемое время (деньги, контракты, долги, планы)

## 📊 Четыре типа времени агента

Для любого агента (человек, фирма, государство, ИИ):

- **t_self** – суверенное время (свобода выбора)
- **t_market** – рыночное время (добровольно продаваемое)
- **t_obligation** – обязательное время (биология + социальные контракты)
- **t_captured** – захваченное время (манипуляция, принуждение, зависимость)

**Баланс**: `t_self + t_market + t_obligation + t_captured = 24 часа/сутки`

## 💰 Богатство и деньги

- **Богатство** = контролируемый объём будущего времени других агентов:  
  `Wealth_J = Σ α_Ji · t_future,i^controlled`
- **Деньги** = стандартизированный контракт на будущее время с коэффициентом доверия α.

## 📈 Ключевые уравнения

- **Власть**: `P_J = Σ α_Ji · (Δt_captured,i – Δt_self,i)`
- **Инфляция**: `π = (ΔM_future – ΔT_real) / ΔM_future`
- **Цена товара**: включает embodied time, ожидаемое рыночное время, качество, доверие.
- **Экономический рост**: `ΔGDP_real = f(Δt_integrated, Δα, Δefficiency)`

## 🧭 Главный индикатор благополучия

**Net Free Time Index (NFTI)** = среднее `t_self – θ·t_captured` по населению.
